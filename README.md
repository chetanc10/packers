# packers
Bash shell wrappers for various archive/unarchive tools

# Archive/Unarchive formats supported
- tar.bz2
- tar.gz
- tar.xz
- tar
- tbz2
- tgz
- rar
- zip
- 7z
- gz
- lzma
- xz
- bz2

# Installation
Using basherbee (recommended): ```basherbee install chetanc10/packers```  
Using basher: ```basher install chetanc10/packers```
